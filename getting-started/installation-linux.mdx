    <Warning>PlexDev.live is an unofficial tool by [**Bali0531**](https://plexdev.live/), where it is not affiliated with this documentation or Plex Development. Please review the code before executing any commands.</Warning>

    To use PlexDev.live, run theses commands:
    <Tabs>
      <Tab title="Stable" icon="shield-check">
      ```bash
      # Quick Installation
      curl -sSL https://plexdev.live/setup.sh | sudo bash

      # or, choose the manual installation:
      git clone https://github.com/Bali0531-RC/plexinstaller.git
      cd plexinstaller
      python3 -m venv .venv && source .venv/bin/activate
      pip install -r requirements.txt
      python installer.py
      ```
      </Tab>
      <Tab title="Beta" icon="test-tube-diagonal">
      ```bash
      # Quick Installation
      curl -sSL https://plexdev.live/setup.sh | sudo bash -s -- -b

      # or, choose the manual installation:
      git clone https://github.com/Bali0531-RC/plexinstaller.git
      cd plexinstaller
      python3 -m venv .venv && source .venv/bin/activate
      pip install -r requirements.txt
      python installer.py
      ```
      </Tab>
